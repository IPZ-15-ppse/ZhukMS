В директории documentation находятся 2 поддиректории, которые содержат сгенерированные jsDoc файлы описания js-скриптов.
Сгенерированные файлы представляют собой набор html-страниц. В index.html каждой поддиректории было вручную добавлено словесное 
общее описание того, что выполняется в этих скриптах.
